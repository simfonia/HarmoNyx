ADSR Envelope

ADSR is a model used to control how a sound changes over time. it defines the volume (or filter) curve from the moment a note starts until it ends.

1. Parameter Definitions

ParameterNameDescription
A (Attack)AttackThe time it takes for the sound to reach its peak level.
D (Decay)DecayThe time it takes for the sound to drop to the Sustain level.
S (Sustain)SustainThe constant volume level while the key is held (0.0 - 1.0).
R (Release)ReleaseThe time it takes for the sound to fade out after the key is released.

2. Common Sound Presets

  • Piano: Very fast attack, moderate sustain, moderate release.
    A: 0.01, D: 0.3, S: 0.4, R: 1.0
  • Strings: Slow attack (fade-in), high sustain, slow release (long tail).
    A: 0.5, D: 1.0, S: 0.8, R: 1.5
  • Drums: Instant attack, fast decay, zero sustain.
    A: 0.005, D: 0.1, S: 0.0, R: 0.1

3. Pro Tip

In HarmoNyx, you can observe the shape of these parameters via the visual chart on the block. A yellow light point will dance according to the playback progress.
