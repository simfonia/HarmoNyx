音訊效果器 (Audio Effects)

效果器能讓聲音產生質變。你可以將效果器放在「定義樂器」容器內來處理特定樂器，或是放在 setup 區塊中作為全域 (Master) 效果。

串接順序非常重要： 效果器是「由上而下」串接的。例如：先放「失真」再放「延遲」，會得到帶著破音的回聲；反之則會得到許多破音的小碎音。

1. 濾波器 (Filter / MoogFilter)

模擬經典的 Moog 合成器低通濾波器，能修飾聲音的明暗度。

  • 頻率 (Frequency) [範圍: 20 - 20000 Hz]：切斷頻率。數值愈低，聲音愈悶。
  • 共振 (Resonance/Q) [範圍: 0.0 - 0.9]：在切斷頻率處產生強調。數值愈高，音色愈尖銳。（若設定大於 1.0 會被限制以防止產生刺耳迴授）。

2. 延遲 (Delay)

產生重複的回聲效果，增加聲音的空間感與厚度。

  • 延遲時間 (Delay Time) [範圍: 0.001 - 5.0 秒]：原始聲音與回聲之間的時間間隔。
  • 回饋 (Feedback) [範圍: 0.0 - 0.95]：回聲循環的次數。（限制在 1.0 以下以避免產生無限遞增的噪聲）。

3. 混響 (Reverb)

模擬在大空間中產生的細密回響，讓聲音更豐滿、更具臨場感。

  • 房間大小 (Room Size) [範圍: 0.0 - 1.0]：空間的開闊程度。
  • 高頻衰減 (Damping) [範圍: 0.0 - 1.0]：模擬材質吸收高頻。數值越高聲音越柔和。
  • 濕度 (Wet) [範圍: 0.0 - 1.0]：殘響效果的大小。0 為純原音，1 為純殘響。

4. 自動濾波 (Auto-Filter)

透過 LFO 自動控制濾波器掃頻，營造規律的頻率演變，產生類似「哇哇 (Wah-wah)」的動態聽感。

  • 速率 (Rate) [範圍: 0.01 - 20.0 Hz]：濾波器擺動的速度。
  • 深度 (Depth %) [範圍: 0 - 100%]：濾波器移動的頻率範圍。
  • 共振 (Q) [範圍: 0.0 - 0.9]：濾波器的尖銳度。

5. 音高調變 (Pitch-Mod)

對音訊的頻率進行動態調變，是製造 Lo-Fi 質感與生命力的核心工具。

  • 調變類型 (Type)：
    • 顫音 (Vibrato)：使用規律的正弦波，聽起來像小提琴的揉弦，具音樂性。
    • 抖動 (Jitter)：使用極速跳動的隨機噪音。聽起來像是一種聲音邊緣的「粗糙質感」。
  • 速率 (Rate) [範圍: 0.1 - 50.0 Hz]：調變的速度。
  • 深度 (Depth) [範圍: 0 - 1200 音分]：
    • 在顫音中，它是音高擺動的半徑（1200 音分代表一個八度）。
    • 在抖動中，它代表「音高模糊的寬度」。用於模擬老舊磁帶馬達轉速不穩。

6. 位元破壞 (BitCrush)

降低音訊採樣精度，製造粗糙、復古的 8-bit 電玩音色（Lo-Fi）。

  • 位元深度 (Bit Depth) [範圍: 1.0 - 16.0 Bits]：解析度越低聲音越破碎。

7. 波形塑形 (Waveshaper)

提供溫暖且具有類比質感的失真效果。透過 tanh 函數讓聲音變得厚實。

  • 失真量 (Amount) [範圍: 0.1 - 100.0]：數值越大越接近方波。

8. 壓縮器與限制器 (Dynamics)

控制音量波動，防止爆音 (Clipping) 並增加聲音的紮實感。

  • 閾值 (Threshold) [範圍: -80.0 到 0.0 dB]：觸發壓縮的音量門檻。
  • 壓縮比 (Ratio) [範圍: 1.0 到 100.0]：壓縮的強度。
  • 啟動/釋放 (Attack/Release) [範圍: 0.001 - 2.0 秒]：反應速度。
  • 增益補償 (Makeup) [範圍: 0.0 到 24.0 dB]：壓縮後的音量補償。
即時更新： 在 draw 迴圈或序列埠事件中使用效果器積木，可以動態改變參數，舞台會即時反映變化。
