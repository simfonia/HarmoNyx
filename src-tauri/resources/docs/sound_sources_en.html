Sound Source Creation

This is the laboratory of sound. In HarmoNyx, all sounds must first be defined here and given an "Instrument Name" before they can be used in performance blocks.

1. Init Minim Audio Engine

The master switch for all audio features. Place this block at the very top of your setup block. It initializes the computer's sound card and prepares output channels.

2. Define Instrument (Container)

A Hat block used to encapsulate all settings for one instrument. You must set the "Name" (e.g., MySynth).

Note: A container can only have one sound source block (Waveform, Sampler, or Synth). If multiple are placed, the lower block will overwrite the previous ones.

3. Set Waveform

Basic subtractive synthesis source. Supports Sine, Square, Triangle, and Sawtooth.

4. Set ADSR

Controls the "energy shape" of the sound. Must be placed inside a "Define Instrument" container.

  • A (Attack): Attack time. Time taken for the sound to reach peak volume.
  • D (Decay): Decay time. Time taken to drop from peak to sustain level.
  • S (Sustain): Sustain level. The volume level held while a key is pressed (0.0 ~ 1.0).
  • R (Release): Release time. Time taken for the sound to fade to silence after the key is released.

5. Harmonic/Additive Synths

Create complex timbres by combining multiple frequency components.

  • Harmonic Synth: Set amplitudes for the fundamental and its overtones.
  • Additive Synth: Customize waveforms, frequency ratios, and amplitudes for each component.

6. Melodic Sampler

Uses recordings of real instruments. The system automatically picks the closest sample and shifts its pitch based on the note played.

ADSR Limitation: To preserve the realistic "Attack" transients (like a piano strike), Samplers skip A, D, and S settings and only apply Release (R) for the tail.

7. Drum Sampler

Designed for drum kits and sound effects. These are "Fixed Pitch"—they play the raw sample regardless of the note played. Recommended for use with Step Sequencers.

8. Mixed Source

Combines a basic oscillator with random noise to create rich, textured tones with an "organic" or "analog" feel.

  • Level: The percentage of noise in the final sound.
  • Jitter: Uses noise to modulate the oscillator's frequency, simulating analog pitch instability.
  • Sweep: Automatic filter modulation. Rate controls the speed (Hz), and Depth controls the frequency range.

Using Custom Samples

HarmoNyx supports loading your own audio files (.wav, .mp3, .aiff).

Quick Open: Click the Settings (Icon) > Open Sound Samples Folder on the right side of the toolbar. The system will automatically open the correct storage location for you.

Using Custom Samples

HarmoNyx supports loading your own audio files (.wav, .mp3, .aiff).

Quick Open: Click the Settings (Icon) > Open Sample Folder on the right side of the toolbar. The system will automatically open the correct storage location for you.

A. Custom Percussion

Select CUSTOM... in the dropdown and enter the path relative to that folder.

Example: If the file is at my_sfx/boom.wav inside the opened folder, enter my_sfx/boom.wav.

B. Custom Melodic Instruments

Melodic samplers require a folder of samples. Create a dedicated sub-directory (e.g., my_piano/) and follow these rules:

  1. Naming: Files must include the pitch name (e.g., C4.wav, Fs3.mp3, Ab5.wav).
  2. Format Support: Use # or s for Sharp notes, and b for Flat notes.
  3. Smart Filling: You don't need every semitone. The system automatically selects the nearest pitch and performs real-time pitch shifting.

Enter the folder name in the path field (e.g., my_piano).

Tip: You can adjust ADSR or waveforms anytime; the stage reflects changes instantly. Glows or warning icons indicate conflicts (like duplicate names or too many sources).
